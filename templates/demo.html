Ghost Font CAPTCHA

No API keys. Drop-in widget. Works on any site.

Live demo

Complete the CAPTCHA, then submit.

Integration

Two lines on the client, one call on the server.

1. Client side

<!-- Load the widget script once -->
<script src="{{ public_base_url }}/api.js" async defer></script>

<!-- Add a div where you want the widget -->
<div class="ghost-captcha"></div>

2. Server side

# Verify the token server-to-server
POST {{ public_base_url }}/api/siteverify
Content-Type: application/json

{ "response": "<ghost-captcha-response value>" }

# 200 OK
{ "success": true, "challenge_ts": "…", "hostname": "…" }
